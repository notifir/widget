wc-notification-bell

A simple notification bell built as a Web Component

                
<script
  type="module"
  src="https://cdn.jsdelivr.net/npm/wc-notification-bell@latest">

This small and simple web component can be consumed by vanilla JS or any front-end framework. It's designed to be integrated with the GraphQL API, similar to Notifications API.

  • Lightweight (~60 kB)
  • Portable
  • Minimal Javascript
  • Easy to customize
